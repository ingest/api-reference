# Retrieve all Events

Returns a list of all events that have been generated by the network.

**URL** : `/events`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : `read_events`

## Success Response

**Code** : `200 OK`

**Response example**

Returns the list of events for the authorized network.
